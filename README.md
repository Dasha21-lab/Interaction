# Interaction - Конструктор тестов

**Interaction** — это веб-приложение, которое позволяет любому пользователю создавать интерактивные тесты без знания программирования. Ты можешь добавлять вопросы различных типов, настраивать правильные ответы и делиться готовыми тестами с друзьями или коллегами.

## Стек технологий

### Клиентская часть (Frontend)
-   **Язык:** TypeScript
-   **Фреймворк:** React
-   **Стейт-менеджмент:** Redux
-   **Роутинг:** React Router
-   **Сборщик:** Vite
-   **Стилизация:** SCSS Modules
-   **Архитектура:** Feature-Sliced Design (FSD)
-   **UI Компоненты:** Storybook
-   **Линтер/Форматтер:** ESLint, Prettier

### Серверная часть (Backend)
-   **Среда выполнения:** Node.js
-   **Фреймворк:** [Express / NestJS / Koa] *— выбери свой*
-   **База данных:** [PostgreSQL / MongoDB / SQLite] *— выбери свою*
-   **Аутентификация:** JWT (самописная реализация)

## 🏗️ Архитектура

Проект организован по методологии **Feature-Sliced Design**. Это обеспечивает четкое разделение ответственности и упрощает масштабирование.

📁 Структура

src/
├── app/          # инициализация
├── entities/     # типы данных
├── features/     # бизнес-логика
├── pages/        # страницы
├── shared/       # общие компоненты и хуки
└── widgets/      # UI-блоки

🧪 Тесты
Планируется: Jest + React Testing Library (покрытие ≥70%).
